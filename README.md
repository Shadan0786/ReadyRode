# ReadyRode

A modern, responsive car rental application built with JavaScript, CSS, and HTML.

## Project Overview

ReadyRode is a user-friendly car rental platform that allows customers to browse, search, and book vehicles for their transportation needs. The application features an intuitive interface with real-time availability checking and seamless booking management.

**Technology Stack:**
- **JavaScript** (49.7%) - Core functionality, booking logic, and interactivity
- **CSS** (30.2%) - Responsive design and modern styling
- **HTML** (20.1%) - Semantic structure and markup

## Features

- 🚗 **Vehicle Catalog** - Browse available cars with detailed information
- 🔍 **Search & Filter** - Find vehicles by type, price, and availability
- 📅 **Date Selection** - Easy-to-use calendar for pickup and return dates
- 💳 **Booking System** - Seamless reservation and checkout process
- 📱 **Responsive Design** - Optimized for desktop, tablet, and mobile devices
- 🔐 **User-Friendly Interface** - Intuitive navigation and clear information display

## Getting Started

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Shadan0786/ReadyRode.git
cd ReadyRode
```

2. Open the project in your web browser:
```bash
# Using a local server (recommended)
python -m http.server 8000
# or use Live Server in VS Code
```

3. Navigate to `http://localhost:8000` in your browser

### Usage

- **Browse Vehicles** - Explore the available car options
- **Search** - Use filters to find vehicles matching your criteria
- **Book a Car** - Select dates and complete your reservation
- **Manage Booking** - View and track your rental details
